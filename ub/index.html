Upward Bound Web Design 2017

Andrew J. Ko, Ph.D. | Associate Professor | University of Washington Information School

Course Description

This course will provide an introduction to careers in web design and computing. However, the course will go beyond basic facts about computing careers, also helping you to develop a better sense of what you have learned about computing already, what you might want to learn in the future, who can help you learn in the future, and how this learning might position you for careers in the future.

Learning Objectives

  • Understand what computing is and is not
  • Recognize which stereotypes about computing careers are and are not true
  • Understand the broad range of industries and domains that apply computing
  • Understand the role of other people in developing interest and skills in learning and doing computing
  • Understand web design as an example of a computing career
  • Construct web sites using modern WYSIWYG tools used by practicing web designers

Grading

  • Pre/post test on computing careers (15 points)
  • Homeworks (60 points, 10 points each)
    • Homework 1 - Your interests
    • Homework 2 - Your interests in computing
    • Homework 3 - Critique
    • Homework 4 - Your mentor
    • Homework 5 - Your goals
    • Homework 6 - Who can help
  • Final web site (25 points)
    • 5 points for each page
    • Each page will be judged for it's completeness, based on my feedback

Expectations

Class attendance is mandatory. If a student is more than 5 minutes late, the student will be considered absent. It is expected all students will follow Upward Bound policies. If you need accommodations of any sort, please communicate with the instructor.
